Ejercicio # 2

¿Es posible ejecutar un programa en java que contenga varias clases con métodos main? 
En caso positivo, ¿cómo se determina el punto de entrada a un programa?

Respuesta:

SI es posible, el punto de entrada o arranque del programa se determina con el IDE, indicando cual debe iniciar,
este tipo de prácticas suele ser util por ejemplo para realizar pruebas o Testing.
